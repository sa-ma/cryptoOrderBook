## Built with

* [React](https://reactjs.org/)
* [Redux](https://redux.js.org/)
* [Redux-Thunk](https://github.com/reduxjs/redux-thunk)

## How to test

* Clone the repo
* Go to the root directory and run `npm install`
* Run `npm test` to run all tests
